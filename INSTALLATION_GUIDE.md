# 🖨️ POS 프린터 간편 설치 가이드

> **비개발자를 위한 쉬운 설치 방법입니다!**

## 🚀 빠른 설치 (추천)

### 1단계: 프로그램 다운로드
1. [GitHub 릴리즈 페이지](../../releases/latest)에서 최신 버전 다운로드
2. `POS_Printer_vX.X.X.zip` 파일을 바탕화면에 압축 해제

### 2단계: 한 번에 설치하기

#### 🥇 방법 1: 배치 파일 실행 (추천)
1. **압축 해제한 폴더**를 엽니다
2. **`간편설치.bat`** 파일을 찾습니다
3. **더블클릭**으로 실행합니다
4. **UAC 창이 나타나면 "예" 클릭**
5. 자동으로 모든 것이 설치됩니다!

#### 🥈 방법 2: PowerShell 스크립트 실행
1. **압축 해제한 폴더**를 엽니다
2. **`build.ps1`** 파일을 찾습니다
3. **우클릭** → **"PowerShell로 실행"** 선택
4. **UAC 창이 나타나면 "예" 클릭**
5. 화면의 안내에 따라 진행합니다

> 💡 **둘 다 같은 결과**: 두 방법 모두 동일하게 작동합니다. 배치 파일이 더 안정적입니다!

### 3단계: 프로그램 실행
1. 빌드가 완료되면 **`POS_Printer_Release`** 폴더가 생성됩니다
2. 해당 폴더에서 **`설치.ps1`** 파일을 **우클릭** → **"PowerShell로 실행"**
3. 바탕화면에 **"POS 프린터"** 아이콘이 생성됩니다
4. 아이콘을 더블클릭하여 프로그램을 실행하세요!

## 🛠️ 상세 설치 과정

### ⚡ 자동 설치 과정에서 일어나는 일들
`build.ps1` 스크립트가 자동으로 다음을 처리합니다:

1. **Python 설치 확인 및 자동 설치** (필요한 경우)
2. **필요한 패키지들 자동 설치**
3. **실행파일(POSPrinter.exe) 생성**
4. **설치 파일들 준비**

### 📋 시스템 요구사항
- **운영체제**: Windows 10 이상
- **메모리**: 4GB RAM 이상
- **저장공간**: 2GB 이상 여유 공간
- **인터넷**: 초기 설정 및 패키지 다운로드용

---

## ⚙️ 초기 설정

### 프린터 연결
1. 프로그램 실행 후 **"설정"** 탭 클릭
2. **USB 프린터**: USB로 연결 후 자동 인식
3. **네트워크 프린터**: IP 주소 입력 (기본 포트: 9100)

### Supabase 데이터베이스 연동
1. **"데이터베이스"** 탭에서 설정
2. Supabase 정보 입력:
   - **URL**: `https://your-project.supabase.co`
   - **API Key**: 프로젝트의 anon 키
   - **Project ID**: 프로젝트 ID
3. **"연결 테스트"** 클릭하여 확인

## 🔧 문제 해결

### ❌ "실행 정책" 오류가 나는 경우
Windows에서 PowerShell 스크립트 실행이 차단된 경우입니다.

**빠른 해결방법**:
1. **`build.ps1` 파일 우클릭** → **"속성"**
2. 하단에 **"차단 해제"** 체크박스가 있으면 **체크 해제**
3. **"확인"** 클릭 후 다시 실행

**또는 PowerShell 명령으로 해결**:
1. **PowerShell을 관리자 권한으로 실행**
2. 다음 명령어 입력:
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
3. `Y` 입력 후 엔터
4. `build.ps1` 다시 실행

### ❌ "관리자 권한이 필요합니다" 오류
**해결방법**:
1. **시작 메뉴**에서 "PowerShell" 검색
2. **우클릭** → **"관리자 권한으로 실행"**
3. 프로그램 폴더로 이동한 후 스크립트 실행

### ❌ Python 자동 설치가 실패하는 경우
**해결방법**:
1. [Python 공식 사이트](https://www.python.org/downloads/)에서 직접 다운로드
2. 설치 시 **"Add Python to PATH"** 체크 필수
3. 설치 완료 후 `build.ps1` 다시 실행

### ❌ 프로그램이 실행되지 않는 경우
**해결방법**:
1. **Windows Defender** 실시간 보호에서 예외 처리
   - 설정 → 업데이트 및 보안 → Windows 보안 → 바이러스 및 위협 방지
   - 바이러스 및 위협 방지 설정 관리 → 제외 항목 추가
   - 프로그램 설치 폴더 전체 제외
2. 다른 백신 프로그램에서도 예외 처리

### ❌ 프린터가 인식되지 않는 경우
**해결방법**:
1. 프린터 전원 및 USB 연결 확인
2. Windows 설정 → 프린터 및 스캐너에서 프린터 상태 확인
3. 프린터 드라이버 재설치
4. USB 포트 변경

## 📞 도움이 필요하세요?

### 자주 묻는 질문

**Q: 빌드 과정이 너무 오래 걸려요**
A: 처음 설치 시 Python과 모든 패키지를 다운로드하므로 10-20분 정도 소요될 수 있습니다. 인터넷 속도에 따라 시간이 달라집니다.

**Q: 여러 대 컴퓨터에서 사용할 수 있나요?**
A: 네! 각 컴퓨터에서 동일한 과정으로 설치하고 같은 Supabase 데이터베이스를 사용하면 됩니다.

**Q: 인터넷 없이도 사용할 수 있나요?**
A: 설치된 후에는 기본적인 프린터 기능은 사용 가능하지만, 데이터베이스 동기화는 인터넷이 필요합니다.

## 🗑️ 프로그램 제거

### 완전 제거 방법
1. 프로그램 종료
2. **제어판** → **프로그램 제거**에서 "POS Printer" 제거
3. 또는 설치 폴더(`C:\Program Files\POS Printer`) 직접 삭제
4. 바탕화면 바로가기 삭제
